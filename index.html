Installation Guide

Tiwut Videos is an open-source Python tool. You need Python and FFmpeg.

1. Install Dependencies

Ensure Python is installed, then install the GUI library via terminal:

pip install customtkinter

2. Setup FFmpeg

Download FFmpeg and add the bin folder to your Windows PATH variables. This is the engine behind the tool.

3. Run the Tool

Save the code as tiwut_videos.py and run it:

python tiwut_videos.py
